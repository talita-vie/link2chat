### ğŸ’¼ The Business Problem
Sales teams and recruiters spend hours manually saving phone numbers into their contacts just to send a first message on WhatsApp. This friction slows down outreach and reduces daily conversion rates.

### ğŸ’¡ The Solution
**Link2Chat** is a custom Chrome Extension built to eliminate this bottleneck. It allows professionals to start a WhatsApp conversation with any number instantly, directly from their browser, without clogging their phone book.

## ğŸ› ï¸ Technical Overview (For Recruiters/CTOs)

This project demonstrates scalable frontend architecture and DOM manipulation within the Chrome Extension ecosystem (Manifest V3).

* **Stack:** React.js, Vite, CSS3.
* **Architecture:** Component-based structure with `useCallback`/`useMemo` for performance optimization.
* **Quality Assurance:** Implemented RegEx sanitization, error boundaries, and accessibility (A11y) standards.
* **Security:** `noopener/noreferrer` implementation to prevent tabnabbing attacks.

## ğŸš€ How to Run Locally

If you are a developer or client wanting to test the code:

1.  Clone the repo: `git clone https://github.com/talita-vie/link2chat.git`
2.  Install dependencies: `npm install`
3.  Build the extension: `npm run build`
4.  Load the `dist` folder into Chrome Extensions in Developer Mode.
